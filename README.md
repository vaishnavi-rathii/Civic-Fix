# CivicFix

A web app where citizens can report civic issues like potholes, garbage, broken streetlights, and water leakage to local authorities.

Built as part of the Design Thinking and Innovation course at Bennett University.

## What it does

- Citizens can register and log in
- Report an issue with photo, description, and location
- View all reported issues in the public feed
- Track status of your issue (Reported, In Progress, Resolved)
- Admin can view all complaints and update their status
- Get notified when your issue status changes

## Tech Used

- React + Vite
- Tailwind CSS
- React Router
- LocalStorage (no backend yet)

## How to Run

1. Download or clone the repo
2. Open terminal in the project folder
3. Run `npm install`
4. Run `npm run dev`
5. Open `http://localhost:5173`
